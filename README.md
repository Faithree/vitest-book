# Vitest 测试指南

这是一个全面的 Vitest 测试教程，包含了从入门到进阶的各种测试场景和最佳实践。

## 🚀 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm docs:dev

# 构建文档
pnpm docs:build

# 预览构建结果
pnpm docs:preview
```
